Стилизация HTML5 плэйсхолдера с помощью CSS

В HTML5 есть замечательный атрибут placeholder, назначение которого - вывести подсказку в поле ввода. Задается он следующим образом: 


  <input type="text" placeholder="текстовая подсказка"/>

Обычно подсказка отображается серым цветом, но иногда возникает необходимость стилизовать этот текст с помощью CSS. На данный момент стандартная спецификация не позволяет нам этого делать, и CSS правила для плэйсхолдеров не стандартизированы. Тем не менее разработчики самых популярных браузеров предоставили нам эту возможность.

Каждый браузер по-своему реализовал поддержку стилизации плэйсхолдеров. В IE и старых firefox (до 18-го) placeholder считается псевдоклассом, а в новых firefox, webkit и blink – псевдоэлементом.

Для стилизации плэйсхолдеров используются следующие правила: 

  • 
  ::-webkit-input-placeholder { /* WebKit browsers */
    //Ваши стили
  }
          
  • 
  :-moz-placeholder { /* Mozilla Firefox 4 to 18 */
    //Ваши стили
  }
          
  • 
  ::-moz-placeholder { /* Mozilla Firefox 19+ */
    //Ваши стили
  }
          
  • 
  :-ms-input-placeholder { /* Internet Explorer 10+ */
    //Ваши стили
  }

Если вы используете CSS-препроцессор, то для применения стилей к плэйсхолдерам можно написать простой миксин. Пример на Sass: 


  @mixin placeholder {
    &::-webkit-input-placeholder { 
      //Ваши стили 
    }
    &:-moz-placeholder           { 
      //Ваши стили 
    }
    &::-moz-placeholder          { 
      //Ваши стили 
    }
    &:-ms-input-placeholder      { 
      //Ваши стили }  
  }

При этом нужно помнить, что совмещать эти селекторы в один нельзя, и если вы напишете: 


  input:-moz-placeholder,
  input::-webkit-input-placeholder { }

то CSS правила работать не будут.

Список поддерживаемых CSS стилей для атрибута placeholder:

  • background
  • color
  • font
  • letter-spacing
  • line-height
  • opacity
  • text-decoration
  • text-indent
  • text-overflow
  • text-transform
  • vertical-align
  • word-spacing

Разные браузеры по разному реализуют сокрытие плэйсхолдера: в одних при получении фокуса инпутом, в других - при вводе первого символа. Второй вариант нередко вводит в ступор людей, которые не знают, что это такое. Они пытаются выделить этот текст и удалить, вместо того, чтобы просто начать печатать. Что ж, это досадное недоразумение с легкостью решается с помощью преведённого выше кода. Достаточно задать плэйсхолдеру прозрачный цвет при фокусе на поле ввода: 


  :focus::-webkit-input-placeholder {color: transparent}
  :focus::-moz-placeholder          {color: transparent}
  :focus:-moz-placeholder           {color: transparent}
  :focus:-ms-input-placeholder      {color: transparent}

Можно также добавить transition для плавного появления и скрытия плэйсхолдера. А вместо прозрачности - задать translate, и плэйсхолдер будет красиво уезжать и скрываться за пределами поля ввода.
